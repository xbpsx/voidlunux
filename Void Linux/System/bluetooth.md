**Для отключения уведомлений** о подключении и отключении Bluetooth-устройств в

`blueman-manager` (который вы, вероятно, используете через `blueman-applet` в окружении Niri), можно воспользоваться двумя основными способами: через графический интерфейс или через терминал с помощью `gsettings`.

1. Через графический интерфейс (GUI)

2. Откройте **Bluetooth Manager** (`blueman-manager`).
3. В верхнем меню выберите **Вид (View)** → **Плагины (Plugins)**.
4. В открывшемся окне найдите плагин **ConnectionNotifier**.
5. Снимите галочку с этого плагина.
    - _Примечание: это полностью отключит всплывающие уведомления о статусе соединений_.

Чтобы блютуз сразу работал, и не нужно было его врубать самостоятельно, нужно

```bash
sudo nano /etc/bluetooth/main.conf
```

Найдите в секции `[Policy]` строку `AutoEnable`.

Установите значение: **`AutoEnable=true`** (убедитесь, что строка не закомментирована символом `#`)

И в настройках `niri` добавить

```kdl
# config.kdl

spawn-at-startup "sh" "-c" "rfkill unblock bluetooth && blueman-applet"
