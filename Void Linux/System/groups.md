
| Название        | Значение                                                                                                                                                                                                                                                                              |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`wheel`**     | Самая важная группа. Дает право выполнять команды от имени суперпользователя через `sudo` или `doas`.                                                                                                                                                                                 |
| **`floppy`**    | Историческая группа для доступа к дисководам гибких дисков (FDD). На современных системах практически не используется.                                                                                                                                                                |
| **`audio`**     | Дает доступ к звуковым устройствам (вывод и ввод звука).                                                                                                                                                                                                                              |
| **`video`**     | Дает доступ к продвинутым функциям видеокарты (GPU) для рендеринга (OpenGL, Vulkan) и аппаратного кодирования/декодирования (VA-API, VDPAU). **Критически важна** для нормальной работы графической среды и игр.                                                                      |
| **`cdrom`**     | Дает доступ к приводам CD/DVD/Blu-ray.                                                                                                                                                                                                                                                |
| **`optical`**   | Дает доступ к приводам оптических дисков (аналогично `cdrom`, иногда используется разными программами).                                                                                                                                                                               |
| **`storage`**   | Дает доступ к подключаемым устройствам хранения данных (USB-флешки, внешние HDD/SSD). Позволяет монтировать и использовать их.                                                                                                                                                        |
| **`input`**     | Дает доступ к устройствам ввода (клавиатура, мышь, планшет, джойстик) для чтения событий на низком уровне. Нужна для переназначения клавиш, управления подсветкой клавиатуры и т.д.                                                                                                   |
| **`kvm`**       | Дает доступ к `/dev/kvm` для аппаратной виртуализации с использованием KVM (Kernel-based Virtual Machine). **Обязательна** для быстрой работы виртуальных машин в QEMU, VirtualBox (с модулем KVM) и Android Emulator.                                                                |
| **`bluetooth`** | Дает право управлять Bluetooth-адаптером.                                                                                                                                                                                                                                             |
| **`_seatd`**    | Дает доступ к демону управления графическими сеансами `seatd`. Это современная замена `libseat`, необходимая для запуска графических сред (особенно Wayland-композиторов, таких как Sway, River) без дополнительных менеджеров вроде `elogind` или `systemd-logind`.                  |
| **`network`**   | Раньше давала право на управление сетевыми интерфейсами (включение/выключение Wi-Fi и т.д.). В современных системах часто не используется напрямую, так как управлением сетью занимается NetworkManager, `dhcpcd` или `iwd` через свои механизмы (например, группу `networkmanager`). |
| **`plugdev`**   | Группа для "подключаемых устройств". Часто используется для доступа к смартфонам, некоторым USB-устройствам, программиторам (например, Arduino через serial).                                                                                                                         |
| **`users`**     | Базовая группа для всех обычных пользователей в системе. Гарантирует, что пользователь — не системный, а "человек".                                                                                                                                                                   |
| **`xbuilder`**  | Специфичная для Void Linux группа. Дает право пользователю запускать `xbps-src` — систему для сборки собственных пакетов из исходного кода (аналогично Arch's AUR или Gentoo's ebuild).                                                                                               |
